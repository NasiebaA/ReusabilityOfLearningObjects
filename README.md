# ReusabilityOfLearningObjects
Reusability Of Learning Objects 

Final year project product files
 
This repository contains the product files of the Final year project, Reusability of learning object. 
The ‘Documents’ folder contains the resources that were used in the in the pedagogical study of the RLOs devoloped. 
The ‘Products’ folder contains the SCORM zip folder as well as the software files of the RLOs developed, in addition to 
the learning materials that were repackaged. Also contains the files of the screencast implemented for this project.
